# D. Бинпоиск на отрезке :)

В данной задаче вам необходимо написать функцию с заданным прототипом:

**bool BinarySearch(const int* begin, const int* end, int target):**

Она возвращает *true*, если на полуинтервале \([begin, end]\) имеется значение *target*.

## Формат ввода
В первой строке дано число \(N \left(1 \leq N \leq 10^5\right)\) — длина массива. На второй строке идут \(N\) целых чисел в порядке неубывания. Гарантируется, что по модулю они не превосходят \(10^9\).

На третьей строке идет единственное число \(Q \left(1 \leq Q \leq 10^5\right)\) — число вызовов функции **BinarySearch**.

Далее идет \(Q\) строк запросов в формате \(i\ j\ t\ (0 \leq i < j \leq N, |t| \leq 10^9)\): \(i, j\) — индексы массива, задающие полуинтервал для вызова функции, \(t\) — аргумент *target* в вызове функции.

## Формат вывода
Для каждого вызова функции выведите YES, если она вернула *true*, и NO — иначе.
