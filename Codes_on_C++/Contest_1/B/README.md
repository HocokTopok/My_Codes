# B. Одинокое число (Medium)

Имеется n уникальных чисел, все кроме двух повторяются дважды. Нужно найти одинокие числа.

## Формат ввода
В первой строке дано число \( n \), \( 1000000 >= n >= 1 \)

В следующей строке даны \( 2n - 2 \) чисел.

## Формат вывода
Два одиноких числа в порядке возрастания.
