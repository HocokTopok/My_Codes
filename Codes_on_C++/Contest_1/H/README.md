# H. Период числа

Дано два числа - \( p \) числитель и знаменатель \( q \). Записать дробь в десятичном виде, если она периодическая, период записать в круглых скобках(см. примеры)

## Формат ввода
Два числа через пробел, \(-100000 \leq p \leq 100000\), \( 1 \leq q \leq 100000 \).

## Формат вывода
Запись числа
