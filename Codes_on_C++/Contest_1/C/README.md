# C. Сможешь посчитать?

Посчитайте следующую сумму \(\sum_{i=1}^n \sum_{j=i}^n a_i \oplus a_j\), где \(\oplus\) – побитовое исключающее ИЛИ.

## Формат ввода
В первой строке входных данных записано число \(n\) (\(1 \leq n \leq 5 * 10^5\)) – количество чисел в последовательности.

Во второй строке записаны \(n\) чисел последовательности \(a_i\) (\(0 \leq a_i \leq 2^{30}\)).

## Формат вывода
В единственной строке выходных данных записано число – ответ на задачу. Так как ответ может быть слишком большим, выведите его по модулю числа \(10^9 + 7\).
