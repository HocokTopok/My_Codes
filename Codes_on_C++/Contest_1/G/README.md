# G. Кратные подмассивы

Имеется массив из \( n \) элементов, необходимо найти число подотрезков, сумма элементов которых кратна \( k \).

## Формат ввода
В первой строке даны два числа \( 1 \leq n \leq 100000, 1 \leq k \leq 100000 \)

Далее во второй строке идут элементы массива, через пробел(\( a_i \leq 1000 \)).

## Формат вывода
Вывести число кратных \( k \) подмассивов.
